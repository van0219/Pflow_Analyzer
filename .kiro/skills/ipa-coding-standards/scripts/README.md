# IPA Coding Standards Scripts

Python scripts for the IPA Coding Standards skill.

## Overview

These scripts implement the stateless pipeline architecture for coding standards analysis.

## Scripts Location

All scripts are located in `ReusableTools/IPA_CodingStandards/`:

- `preprocess_coding_standards.py` - Phase 0
- `build_naming_analysis.py` - Phase 1
- `build_javascript_analysis.py` - Phase 2
- `build_sql_analysis.py` - Phase 3
- `build_errorhandling_analysis.py` - Phase 4
- `assemble_coding_standards_report.py` - Phase 6

# IPA JavaScript ES5 Analyzer - References

This directory contains reference materials for the IPA JavaScript ES5 analyzer skill.

## Files

### es5-compliance-rules.md
Comprehensive ES5 compliance rules, forbidden ES6+ features, and conversion patterns.

### best-practices.md
IPA-specific JavaScript best practices, defensive programming patterns, and production-ready code guidelines.

### common-issues.md
Common JavaScript issues found in IPA processes, with examples and fixes.

## Usage

These reference files are loaded automatically by the skill when analyzing JavaScript code. They provide:

- ES5 syntax rules and constraints
- Conversion patterns from ES6+ to ES5
- Best practices for IPA JavaScript
- Common anti-patterns and fixes
- Production-ready code patterns
